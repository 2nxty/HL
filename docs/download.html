  About Antivirus False Positives

Why does VirusTotal show detections?

HyLauncher is not digitally signed with a Microsoft certificate (which costs hundreds of dollars annually). Because of this, some antivirus engines flag it as potentially unwanted software, resulting in approximately 2 false positives on VirusTotal.

Windows SmartScreen warning?

Windows may show a "Windows protected your PC" message saying the app is from an unknown publisher. This is normal for unsigned software. Simply click "More info" and then "Run anyway" to proceed.

Is it safe? Yes. The majority of antivirus engines recognize HyLauncher as clean. The false positives are due to the lack of code signing, not actual malware.

If you're concerned, you can always scan the downloaded file yourself or run it in a sandboxed environment first.
